Test environment:
OS: macOS 10.13.6 / Ubuntu 16.04
Java: 1.8.0_131
Tomcat: 8.5.15
MySQL: 5.7.18
Maven: 3.5.0

How to run:
1. set up database:
    1.1 create new database name SirenStore;
    1.2 restore the database use the backup file SirenStore.sql.  $ mysql -u USERNAME -p SirenStore < SirenStore.sql
    1.3 go to /src/main/resources/applicationContext.xml config your datasource
2. install the application to tomcat:
    2.1 use maven to package up the .war file.  $ mvn packge
    2.2 copy the /target/SirenStore.war to %Tomcat_Home%/webapps
    2.3 start tomcat
3. open your browser and visist localhost:8080/SirenStore, the port number is based on your tomcat setting
